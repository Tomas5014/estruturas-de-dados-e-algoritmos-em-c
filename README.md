# 📚 Estruturas de Dados e Algoritmos em C

Implementações de estruturas de dados clássicas desenvolvidas durante a disciplina de **Estrutura de Dados** na graduação. Cada estrutura foi codificada do zero em C, com foco na compreensão dos mecanismos internos, complexidade de tempo e gerenciamento manual de memória.

---

## 🗂️ Estruturas Implementadas

### 🌳 Árvore (`/arvore`)
Implementação de estrutura de dados em árvore, explorando hierarquia, busca e travessias.

### 📊 Grafo (`/grafo`)
Representação e manipulação de grafos, estrutura fundamental para modelagem de redes, caminhos e relacionamentos.

### #️⃣ Tabela Hash (`/hash`)
Estrutura de mapeamento chave-valor com tratamento de colisões, viabilizando buscas em tempo médio O(1).

### 🔺 Heap (`/heap`)
Implementação de heap (min ou max), estrutura base para filas de prioridade e algoritmos de ordenação como o Heapsort.

---

## 🛠️ Tecnologias

- **Linguagem:** C (C99)
- **Compilador:** GCC

---

## ▶️ Como compilar e executar

Clone o repositório:

```bash
git clone https://github.com/Tomas5014/estruturas-de-dados-e-algoritmos-em-c.git
cd estruturas-de-dados-e-algoritmos-em-c
```

Compile qualquer um dos arquivos com GCC:

```bash
gcc -o programa arvore/arvore.c
./programa
```

> Substitua `arvore/arvore.c` pelo caminho do arquivo que deseja executar.
